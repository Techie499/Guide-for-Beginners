## Reference Variable

Reference Variable vs Primitive Variable
- memory
- assignment
- initialization (null)

Predefined Reference types in Java
- String
- Wrapper Class
- LocalDate, LocalDateTime

Reference variable is stored in Heap. Primitive variable is stored in Stack

>> 1+2
-> 3

>> "2"+"2"
-> "22"

>> "1"+3
-> "13"
